# Node.js의 시작

2008년 구글에서 크롬 웹 브라우저에서 구동되는 자바 스크립트의 성능 개선을 위해 V8 자바스크립트 엔진을 개발한다. Ryan Dahl은 해당 엔진을 기반으로 Node.js를 개발하였다. Node.js는 웹 브라우저를 넘어 컴퓨터 그 자체를 제어할 수 있다. 이를 이용하여 웹 어플리케이션을 제작할 수 있게 되었다.
